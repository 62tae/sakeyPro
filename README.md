환영합니다!!

이 프로젝트는 초보들의 MEAN Stack 협업 프로젝트입니다.

MEAN Stack 예제들을 통해서 진행하고 있으며, Github을 사용한 협업을 테스트하고 있습니다.

# 1. Node & npm 설치

프로젝트를 진행하기 위해서 몇가지 설치해야 할 것들이 있습니다.

https://nodejs.org/en/
